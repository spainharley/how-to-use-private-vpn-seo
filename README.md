## FAQ

**1) How do I confirm the VPN is actually on?**  
Connect, then check your public IP in a browser. If it changes to the server region you picked, traffic is going through the VPN.

**2) When should I use the Kill Switch?**  
Always. It prevents accidental traffic leaks if the tunnel drops during Wi‑Fi switches or brief ISP hiccups.

**3) What server should I choose for best speed?**  
Start with the closest location. If it’s congested, pick another nearby server before jumping across continents.

**4) Does this help on public Wi‑Fi?**  
Yes. With **AES-256 encryption** enabled and a **stable connection**, it prevents local network snooping on open hotspots.
